# 🎙 Wsool STT — التحدث بدلاً من الكتابة

**أداة تحويل الصوت إلى نص لنظام Windows** — اضغط اختصاراً، تحدث، ويظهر النص تلقائياً في أي مكان يكون فيه المؤشر.

> تدعم: Groq · Gemini · OpenAI · النماذج المحلية

---

## لماذا هذا البرنامج؟

**ويندوز لا يدعم اللغة العربية في التعرف على الصوت.**

ميزة الإملاء المدمجة في Windows تعمل مع الإنجليزية والفرنسية والإسبانية والألمانية والصينية وغيرها — لكن **العربية غائبة تماماً**.

Wsool STT يحل هذه المشكلة باستخدام نماذج ذكاء اصطناعي متقدمة (Whisper, Gemini) تفهم العربية بشكل ممتاز، وتلصق النص مباشرةً في أي تطبيق — المتصفح، Word، واتساب، أي شيء.

---

## المميزات

- **يعمل في كل مكان** — أي تطبيق، أي حقل نصي
- **عربي وإنجليزي** — كشف تلقائي للغة أو اختيار يدوي
- **إيقاف تلقائي** — يتوقف عند الصمت
- **وضع الإملاء المستمر** — اضغط الاختصار مرتين للدخول في وضع يسجل باستمرار
- **متعدد المزودين** — Groq، Gemini، OpenAI
- **مجاني تماماً** — انتقال تلقائي عند الوصول لحد الاستخدام المجاني
- **يعمل في الخلفية** — في شريط النظام، لا نافذة، لا terminal

---

## البدء السريع

### 1. احصل على مفتاح API مجاني

| المزود | الحد المجاني | السرعة | الرابط |
|--------|-------------|--------|--------|
| **Groq** | 7,200 ثانية/يوم | الأسرع | [console.groq.com](https://console.groq.com/keys) |
| **Google Gemini** | 1,500 طلب/يوم | سريع | [aistudio.google.com](https://aistudio.google.com/app/apikey) |
| **OpenAI** | مدفوع | سريع | [platform.openai.com](https://platform.openai.com/api-keys) |

### 2. التثبيت

```bash
git clone https://github.com/moazzyadah/Wsool-SST.git
cd Wsool-SST
install.bat
```

### 3. أضف مفتاح API

انسخ `.env.example` إلى `.env` وأضف مفتاحك:

```env
GROQ_API_KEY=ضع_مفتاحك_هنا
# أو
GEMINI_API_KEY=ضع_مفتاحك_هنا
```

### 4. التشغيل

```
run.bat           # مع نافذة (للتجربة)
run_silent.vbs    # في الخلفية بدون نافذة
startup.bat       # إضافة للبدء مع Windows تلقائياً
```

## اختصارات لوحة المفاتيح

| الاختصار | الوظيفة |
|----------|---------|
| `Ctrl+Alt+Space` | بدء / إيقاف التسجيل |
| `Ctrl+Alt+Space` × 2 | تفعيل وضع الإملاء المستمر |
| `Ctrl+Alt+L` | تبديل اللغة (تلقائي ← عربي ← إنجليزي) |
| كليك يمين على الأيقونة | اللغة · الخروج |

### وضع الإملاء المستمر
اضغط `Ctrl+Alt+Space` مرتين متتاليتين للدخول في وضع الإملاء المستمر — يسجل باستمرار، وبعد كل صمت يُفرّغ النص ويبدأ من جديد تلقائياً. حرّك الماوس للخروج من هذا الوضع.

## أداة الإعداد (واجهة رسومية)

شغّل المعالج التفاعلي:

```
venv\Scripts\python.exe installer.py
```

يرشدك خلال اختيار المزود، اختبار المفتاح، اللغة، وتخصيص الاختصارات.
